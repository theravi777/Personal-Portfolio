# 💼 Personal Portfolio

A modern, responsive personal portfolio website built using **HTML, CSS, and JavaScript** to showcase my skills, projects, education, and contact information. It features a clean UI, smooth animations, and an interactive user experience.

## 🚀 Features

- Responsive design
- Interactive animations
- Custom cursor & particle background
- Smooth scrolling
- Project showcase
- Animated skills section
- Contact section

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
